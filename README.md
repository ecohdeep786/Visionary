Development
-----------

This repo uses a workspace layout. To run the Next.js app locally:

```bash
# install a package manager that supports workspaces (pnpm recommended)
pnpm install
pnpm --filter visionary-web dev
```

Or run from the app folder:

```bash
cd apps/web
npm install
npm run dev
```
